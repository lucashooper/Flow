# Lucas's Orbit

A small, personal site for notes, thoughts, music and projects.

## Tech

- Vite
- React + TypeScript
- Plain CSS, using Inter as the primary font

## Running locally

```bash
cd "Lucas's Orbit"
npm install
npm run dev
```

Then open the printed localhost URL in your browser.

## Structure

- **Journal** – chronological entries with titles & blurbs
- **Thoughts** – short one-line ideas
- **Music** – track list with mood + length (hook this up to real audio later)
- **Projects** – quick overview of things you are building

The styling aims for a calm, premium dark UI, loosely inspired by idea-centric sites like Vitalik Buterin's, but customized for Lucas.
